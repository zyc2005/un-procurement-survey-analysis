# Procurement Survey Mapping and Analysis

This repository contains Python code and research materials for a procurement-related survey analysis project. The project focuses on visualizing the geographic distribution of surveyed companies in China and supporting research on procurement, export development, UN-related procurement challenges, and sustainability needs.

## Project Overview

The main goal of this project is to support a feasibility study on establishing a knowledge-sharing, capacity-building, and support services hub in Qingpu District, Shanghai.

The project includes:

- Mapping the distribution of surveyed companies across China
- Visualizing company locations using Python
- Supporting survey-based research on procurement and export challenges
- Reviewing existing procurement, supply chain, and sustainability-related programs in China
- Organizing materials related to UN procurement and sustainable procurement practices
